* R language (sets library)
* and GNU octave

to use AI = fuzzy sets to describe the formulas rather than with simulations.

> 
> ℹ️ fuzzy sets can be thought here as a research tool
> 
> ℹ️ to combine it with the mathematical formulas

Covers many topics:
